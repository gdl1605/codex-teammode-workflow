# Release Notes Draft

## v0.1.0 Public Preview

Codex-teammode Workflow is a prompt-first Team Mode workflow package for Codex and Claude Code.

This release includes:

- a Chinese-first workflow kernel
- synchronized `AGENTS.md` and `CLAUDE.md` entry files
- a bootstrap prompt for installing into target repos
- a docs scaffold for product, architecture, handoff, plans, and evidence
- audit-first and docs impact check rules
- an optional Leader-driven Team Mode flow with planner, generator, scout, and evaluator roles
- an installer with `--dry-run`, `--force`, and `--no-clipboard`

This is an unofficial community project and is not affiliated with, endorsed by, or sponsored by OpenAI.

Recommended first issues after publishing:

- translate the workflow kernel to English
- add generic sanitized install examples
- add CI for shell syntax, markdown links, and secret scanning
- collect Codex and Claude Code compatibility feedback
